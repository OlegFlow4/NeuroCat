# NeuroCat
A simple and aesthetic local AI chatbot based on Gemma 2, written in Python
# 🐱 NeuroCat v1.2 Release

Аккуратный, эстетичный локальный ИИ-ассистент прямо в вашем терминале. Работает на базе модели **Gemma 2 9B** и оформлен в виде умного консольного кота.

## 🌍 Кроссплатформенность и поддерживаемые ОС

NeuroCat написан на чистом Python с использованием универсальных библиотек, поэтому он работает на любой современной операционной системе "из коробки".

* 🐧 **Linux:** Любой современный дистрибутив (Arch Linux, Ubuntu, Fedora и др.). Работает идеально и нативно.
* 🪟 **Windows:** Windows 10 / Windows 11. (Если при установке библиотек возникает ошибка, убедитесь, что у вас установлен C++ компилятор, либо используйте Python 3.12 для загрузки готовых бинарных пакетов).
* 🍏 **macOS:** macOS 11 и новее. На процессорах Apple Silicon (M1/M2/M3) работает особенно быстро благодаря встроенной оптимизации движка `llama.cpp` под графику Metal.

**Требования к ПО:**
* Установленный **Python 3.8** или выше (рекомендуются версии **3.10 — 3.12**).

## ✨ Особенности
* 🧠 **Контекстная память:** Кот хранит историю диалога и не забывает, о чем шла речь в прошлых сообщениях.
* 🎨 **Красивое оформление:** Полная подсветка синтаксиса кода, списков и разметки в терминале благодаря интеграции `rich`.
* 📦 **Умная автоустановка:** Автоматическая фоновая доустановка всех необходимых Python-библиотек (`psutil`, `rich`, `requests`, `llama-cpp-python`) с цветным прогресс-баром при первом запуске.

## 🚀 Запуск
1. Склонируйте этот репозиторий.
2. Разархивируйте в папку загрузки Download/Загрузки
3. Откройте neurocat.py и ожидайте некоторое время для загрузки Gemma на ваш ПК(время зависит от вашей скорости интерента)

## 💻 Системные требования

Так как нейросеть запускается полностью локально на вашем процессоре или видеокарте, для её работы требуются определенные ресурсы:

### Минимальные:
* **Оперативная память (RAM):** 8 ГБ (при этом желательно закрыть тяжелые вкладки в браузере, так как модель займет около 5.5 ГБ).
* **Процессор (CPU):** Любой современный x86-процессор с поддержкой инструкций **AVX2** (инструкции необходимы для адекватной скорости работы `llama.cpp`).
* **Место на диске:** ~6 ГБ свободного пространства.

### Рекомендуемые (для быстрой генерации):
* **Оперативная память (RAM):** 16 ГБ или больше.
* **Видеокарта (Видеопамять):** Дискретная графика с **8 ГБ VRAM** (например, RTX 3060 / 4060) с поддержкой CUDA. В скрипте включен автоматический перенос всех слоев модели в видеопамять (`n_gpu_layers=-1`), что ускорит ответы в разы.
* **Тип диска:** **SSD** (на обычном жестком диске HDD модель будет загружаться в память при старте скрипта около 2–3 минут, на SSD — пару секунд).
